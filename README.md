# REX-47 Firmware

## Overview
This repository contains the low-level embedded firmware for the REX-47 robotic platform. It bridges the gap between the high-level software stack and the physical hardware, including motor controllers, sensor interfaces, and power management systems.

## Key Features
- **Motor Control:** Precise actuation of wheels, arms, or joints.
- **Sensor Polling:** Real-time data acquisition from IMUs, encoders, and environmental sensors.
- **Safety Systems:** Hardware-level emergency stop and fail-safes.
- **Communication:** Serial/CAN bus communication with the primary compute unit.

## Getting Started
Refer to the documentation within for flashing the firmware to the designated microcontrollers.
